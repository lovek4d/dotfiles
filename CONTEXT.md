# Dotfiles

Personal-use utilities and dotfiles for setting up and operating local development machines. This repo optimizes for one user's workflow, not a stable public interface.

## Language

**Agent worktree**:
A disposable Git worktree paired with a tmux session that launches a coding agent against one branch.
_Avoid_: Claude worktree, Codex worktree

**Machine bootstrap**:
Bringing a fresh local development machine up to the baseline expected by these dotfiles.
_Avoid_: setup, install script

**Whisper transcription**:
Capturing local speech, transcribing it with whisper-cli, and delivering the resulting text to the active workflow.
_Avoid_: Hammerspoon transcription
